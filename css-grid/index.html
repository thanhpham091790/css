The place-content Property

Use the place-content property to both vertically and horizontaly align the grid inside the container.

The value "center" positions the grid items in the middle of the container (both vertically and horizontally):

Grid Layout

A Grid Layout must have a parent element with the display property set to grid or inline-grid.

Direct child element(s) of the grid container automatically becomes grid items.
